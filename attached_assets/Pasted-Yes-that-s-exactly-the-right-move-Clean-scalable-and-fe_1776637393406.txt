Final pattern (use this everywhere)

Header structure

Left side:

viva.   Care
viva.   Clinic
viva.   Analytics

* “viva.” = logo (consistent everywhere)
* “Care / Clinic / Analytics” = product context label

⸻

Design rules (important)

1. Keep hierarchy clear

* viva. → primary brand (bolder)
* Care / Clinic / Analytics → secondary (slightly lighter)

Example:

* viva. → bold / darker
* Analytics → medium weight / slightly muted

⸻

2. Same spacing everywhere

* small horizontal gap between logo and label
* consistent across all 3 products

⸻

3. Same placement

* top-left corner
* inside header bar
* identical positioning across:
    * Viva Care
    * Viva Clinic
    * Viva Analytics

Consistency = polish

⸻

4. No dividers or overdesign

Don’t add:

* pipes (|)
* slashes (/)
* boxes
* badges

Keep it minimal:

viva. Analytics

⸻

Behavior across products

Viva Care

viva. Care

Viva Clinic

viva. Clinic

Viva Analytics

viva. Analytics

⸻

Optional (nice but not required)

Subtle visual distinction per product

You can lightly differentiate without breaking brand:

* Care → slightly softer tone
* Clinic → neutral
* Analytics → slightly denser / darker UI

But keep:

* same base palette
* same logo

⸻

What NOT to do

Don’t:

* create separate logos (VivaCare, VivaAnalytics, etc.)
* change colors per product drastically
* restyle the logo per surface
* treat them as separate brands
