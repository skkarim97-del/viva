Worklist row improvement:
Each escalated/review-needed row should clearly show:

1. Why surfaced
Examples:
- “Nausea ↑ + appetite ↓”
- “No check-in 3 days + energy trending down”
- “Severe nausea reported”
- “Patient requested review”

2. Viva support shown
Examples:
- “Viva: small meals + hydration”
- “Viva: bland foods + steady fluids”
- “Viva: rest + hydration support”

3. Patient response
Examples:
- “Patient: tried → better”
- “Patient: tried → worse”
- “Patient: awaiting feedback”
- “Patient: not tried yet”

4. Action label
Use:
- “Acknowledge & engage”

Avoid:
- “call”
- vague “review needed” without context
- long paragraphs

Recommended row format:
Line 1:
[patient name] [category chip] [review now]

Line 2:
Nausea ↑ + appetite ↓

Line 3:
Viva: small meals + hydration · Patient: awaiting feedback

Line 4/action:
Acknowledge & engage · No recent engagement

Keep the row compact. Do not make it visually heavy.

Visual guidance:
- Keep existing layout and style

- Use existing severity/risk colors
- Red only for true severe/urgent states
- Orange/amber for review-needed states
- Do not clutter the row with too many badges
